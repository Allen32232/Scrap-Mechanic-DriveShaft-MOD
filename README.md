## 📖 简介

《DriveShaft - 传动轴》模组为《Scrap Mechanic》带来了一种全新的动力传输方案。  
通过视觉化的传动轴特效，你可以清晰、动态地观察动力传递路径，轻松搭建既美观又高效的机械作品。  
本模组旨在简化旋转结构的搭建过程，让创意不再受限于复杂的轴承堆叠。

---

## ✨ 特性

- **视觉化传动轴**：实时显示动力流动，支持自定义颜色、粗细与特效强度。
- **三种核心逻辑零件**：节点、控制器、速度传感器，组合灵活，易于上手。
- **动态调节**：通过控制器面板即可调整外观，无需重新搭建。
- **广泛兼容**：可与轮胎、发动机、螺旋桨等任何旋转部件配合使用。

---

## 🧩 模组内容

### 1. 传动轴节点
- **作用**：定义传动轴的起点与终点。
- **用法**：
  1. 在两个需要连接的位置分别放置节点。
  2. 使用游戏内的**逻辑连接线**将两个节点相互连接，以确定传动轴的路径。

### 2. 传动轴控制器
- **作用**：传动系统的核心控制单元。
- **用法**：
  1. 将任意一个已连接的节点用逻辑线连接到控制器，即可激活整条传动轴。
  2. 按 `E` 键打开设置面板，实时调整：
     - 传动轴颜色
     - 粗细
     - 特效强度

### 3. 传动轴速度传感器
- **作用**：为传动轴提供旋转驱动信号。
- **用法**：
  1. 将传感器安装在任意旋转部件上（如轮胎、发动机转子、螺旋桨等）。
  2. 用逻辑线将传感器连接到传动轴控制器。

## 📦 安装方法

### 通过 Steam 创意工坊（推荐）
1. 确保已安装《Scrap Mechanic》并登录 Steam。
2. 访问创意工坊页面并订阅本模组：[点击此处订阅]（链接待补充）。
3. 启动游戏，模组将自动下载并启用。

### 手动安装
1. 从 [Releases 页面]（链接待补充）下载最新版本的压缩包。
2. 解压到游戏根目录下的 `Mods` 文件夹中，例如：
   - Windows: `C:\Program Files (x86)\Steam\steamapps\common\Scrap Mechanic\Mods\`
   - 确保文件夹结构为 `Mods/DriveShaft-Mod/`（包含 `Scripts` 等子文件夹）。
3. 启动游戏，在 Mods 菜单中手动启用该模组。

## 🛠️ 使用方法

1. **放置节点**：在需要传递动力的两个位置放置传动轴节点。
2. **连接节点**：使用逻辑连接器将两个节点相连。
3. **添加控制器**：放置传动轴控制器，并用逻辑线连接至任意一个节点。
4. **安装速度传感器**：将传感器安装在旋转部件上，并连接至控制器。
5. **调整效果**：对着控制器按 `E` 键，自定义颜色、粗细与特效强度。
6. **启动旋转**：驱动旋转部件（如给发动机供电），传动轴将开始显示动态特效。

---

## ⚠️ 注意事项

- 确保所有逻辑连接正确且无断裂。
- 速度传感器**必须安装在正在旋转的部件上**才能产生有效信号。
- 控制器与节点需处于同一逻辑网络中（通常无需额外供电，但若使用开关，请确保信号流通）。
- 若传动轴不显示，请检查：
  - 节点间逻辑连线是否建立；
  - 控制器是否收到速度传感器的信号；
  - 旋转部件是否实际在转动。
- 目前每个控制器仅支持**一对节点**组成的单条传动轴。如需多段传动，可分别搭建多组系统。

## ❓ 常见问题

**Q: 传动轴不旋转/不显示特效？**  
A: 请按注意事项中的步骤排查连接与信号来源。

**Q: 可以连接多个节点形成复杂网络吗？**  
A: 当前版本仅支持两个节点之间的一条传动轴。若需多段传动，请重复搭建“节点对+控制器”系统。

**Q: 控制器设置面板无法打开？**  
A: 确保你正对着控制器，并且在生存模式中拥有建造权限（或开启管理员模式）。
